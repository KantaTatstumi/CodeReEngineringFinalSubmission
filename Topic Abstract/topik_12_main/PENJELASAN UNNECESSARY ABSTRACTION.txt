Pada code before yang diberikan, ditemukan bahwa untuk setiap warna button yang ada, dibuatkan class khusus untuk menanganinya. Hal ini menjadi sebuah smell yaitu unnecessary abstraction karena sebenarnya class tersebut tidaklah dibutuhkan. Hal ini terlihat pada utilitas class yang ternyata hanya untuk mengganti warna background saja. 

Oleh karena itu, untuk menangani permasalahan ini, solusi yang dilakukan adalah dengan memindahkan isi statement class button yang ada ke class main button saja. Jadi, kita tinggal memanggil setBackground saja saat kita ingin set warna dari background, jika ada kondisi khusus, maka bisa tinggal pakai if. Pada intinya, penyelesaiannya adalah dengan menghapus class tersebut, dan memanggil isi dari class tersebut secara langsung di class mainnya saja karena statementnya hanya ada 1 sehingga tidak butuh dibuatkan class khusus.
